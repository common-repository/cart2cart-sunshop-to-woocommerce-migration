== Description ==
**Supported WooCommerce versions: 1.1.x – 1.6.x, 2.x, 3.x**
**Supported SunShop versions: 4.x.x**

Cart2Cart: SunShop to WooCommerce Migration plugin has been developed to make your migration process easier and faster. Don’t worry if you are not a programmer as the procedure is fully automated including a few steps. Moreover, your SunShop is not going to be downtimed even while the migration proceeds. Therefore, be sure that the step to a brand new powerful store can be effortless and swift.

**The plugin offers 2 migration options**

1. Free Demo migration — lets you perform migration of products, orders, customers and categories in limited quantity (up to 10 entities each) for test purposes.
1. Full Migration — lets you perform UNLIMITED migration of products, orders, customers, categories, variants, attributes and other related entities. Full Migration price starts at $69 and may vary depending on the number of transferable entities and additional options chosen.

= Cart2Cart migration possesses such features =
- **Simple** - Migration Wizard provides a few steps to perform the switch.
- **Fast** - a few hours are needed to get WooCommerce store.
- **Free Demo** - try the service functionality and move up to 10 entities.
- **Support** - ask any questions via phone, chat or email.

= You can transfer the following items from SunShop to WooCommerce =
* Products, product images, product extra fields, product attributes;
* Orders, order statuses;
* Categories, category images;
* Customers, customer shipping address, customer billing address.

**Please note.** *Migration is performed only for database, but not for the template.*

Cart2Cart: SunShop to WooCommerce Migration plugin is a great tool to set an accurate interaction between your Source and Target Carts for data transfer. After your Connection Bridges have been installed automatically, wait until you’ll be redirected to Cart2Cart Migration Wizard. Then there will be a possibility to continue Full Migration just in a couple of steps. Cart2Cart Migration Plugin is free to download and install, but Full Migration is paid and starts from $69.

= Steps to take before migration =
* Perform a WooCommerce store installation. Make sure your stores are available online.
* Prepare SunShop FTP details because they will be required for further Connection Bridges installation.

== Installation ==
1. Get the plugin archived file.
1. Unzip plugin file to your PC.
1. Move extracted files to wp-content/plugin directory.
1. Go to Admin -> Plugins, click “Cart2Cart SunShop to WooCommerce Migration” in order to activate it.
1. Provide SunShop FTP details and afterwards download the Connection Bridge.
1. The Connection Bridge for your WooCommerce store will be uploaded automatically.
1. Click Start Migration button to get a redirection to Cart2Cart page and run Full migration.


== Frequently Asked Questions ==
= What is Cart2Cart? =

Cart2Cart is a shopping cart migration service which allows automated data transfer from one platform to another. Currently service supports migration from/to [70+ most popular shopping carts](http://www.shopping-cart-migration.com/supported-carts). Main beneficial sides for e-merchants are:

* No technical skills required. Since the process is fully automated, there is no need to have high level of programming skills. Intuitive [migration Wizard](http://www.shopping-cart-migration.com/quick-demo-tour) will help to perform a straightforward migration.
* Cart2Cart requires no software installation. The migration is performed on Amazon Elastic Cloud servers. The data migration is performed with the help of connection bridge, which is installed for source and target shopping carts and makes the integration possible.

* Cart2Cart provides a possibility to migrate products, customers,orders with all corresponding data to a desirable shopping cart.

* Flexible pricing system allows to pay only for entities, which are really migrated. The price of migration can be calculated with the help of [Online Estimator](https://app.shopping-cart-migration.com/estimator).

* Totally free Demo migration which provides a possibility to try how everything works before launching a Full migration.
If you have some questions, [Support Team](http://support.magneticone.com/index.php?/Tickets/Submit/RenderForm/5) is ready to answer all questions via live chat, phone or ticket at any convenient time.

= Can I be sure of data security Cart2Cart service? =
Cart2Cart performs migration using a separate Amazon EC2 server and all data is deleted after the process has been finished.
Cart2Cart does not migrate client’s credit card information and passwords.
We use secure HTTPS protocol for all connections and only authorized staff has access to the migration process.

= Do you migrate multiple languages? =
Yes, Cart2Cart supports migration of multiple languages from X-Cart to WooCommerce providing a paid WPML module is installed and configured on your WordPress before the data transfer. Please, [read our FAQ](https://www.shopping-cart-migration.com/faq/45-woocommerce/385007-is-multilanguage-migration-to-from-woocommece-possible) for more info.

= Do you migrate the customer passwords? =
"I would like to migrate from one shopping cart to another with encrypted customer passwords. Is there a way for customers to preserve the accounts but be prompted to enter new passwords? "
Customer accounts can be migrated from source to target shopping cart. Cart2Cart doesn't migrate the passwords, because they are encrypted specifically.
Your clients should use Password Recovery tool on your target shopping cart to access their accounts. However, order history will be available after the migration.
You can find more detailed information about password recovery in our [article](http://www.shopping-cart-migration.com/blog/44-migration-service-tips/20012-box-of-tricks-password-recovery).
